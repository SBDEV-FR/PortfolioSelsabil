Accueil

Application web simple ( HTML, CSS, JS )



Notions mises en pratique :


  • Manipulation du DOM
    • Sélection d'éléments (getElementById, querySelector)
    • Création d'éléments (createElement)
    • Modification d'éléments (innerHTML, textContent)
    • Ajout et suppression d'éléments (appendChild, removeChild)
  • Gestion des événements
    • Ajout d'écouteurs d'événements (addEventListener)
    • Gestion de différents types d'événements (click, keypress, blur)
  • Fonctions
    • Déclaration et appel de fonctions
    • Fonctions anonymes et fléchées
  • Variables et portée
    • Utilisation de let et const
    • Portée des variables dans les fonctions
  • Manipulation de chaînes de caractères
    • Méthodes de chaînes (trim)
  • Structures conditionnelles
    • Utilisation de if pour vérifier des conditions
  • Manipulation de classes CSS
    • Ajout, suppression et basculement de classes (classList.add, remove, toggle)
  • Travail avec les tableaux
    • Gestion de la liste implique des concepts de tableau
  • Gestion de l'état de l'application
    • Maintien de l'état des tâches (ajout, édition, suppression)
  • Programmation événementielle
    • Réaction aux actions de l'utilisateur pour mettre à jour l'interface
  • Manipulation des attributs
    • Modification des attributs des éléments HTML
  • Gestion du focus
    • Utilisation de focus() pour améliorer l'expérience utilisateur lors de l'édition
  • Propagation des événements
    • Compréhension implicite de la propagation des événements dans le DOM
  • Closures
    • Utilisation implicite dans les gestionnaires d'événements
  • Méthodes de chaînage
    • Enchaînement de méthodes pour des opérations successives
  • Manipulation dynamique du contenu
    • Mise à jour en temps réel de l'interface utilisateur

Objectifs

Créer une application web simple pour apprendre le language Javascript. Permet de créer, modifier, supprimer une tâche.
